# Bank Management System (Java - Programming Fundamentals)

## 📌 Description

The Bank Management System is a console-based application developed in Java as part of my Programming Fundamentals (PF) course. The project simulates the basic operations of a banking system through a menu-driven interface, allowing users to register, log in, and perform common banking transactions.

The main objective of this project was to strengthen my understanding of core programming concepts by applying them to a real-world banking scenario.

---

## ✨ Features

### 👤 User Registration
- Create a new bank account
- Enter personal information during registration
- Store user details for future login

### 🔐 Login System
- Secure login using registered credentials
- Separate login options for:
  - User
  - Admin

### 💰 Banking Operations
After logging in as a user, the following operations are available:

- Deposit money
- Withdraw money
- Transfer money to another account
- Apply for a loan
- Check account information (if implemented)
- Return to the main menu
- Exit the application

### 👨‍💼 Admin Access
- Login as administrator
- Access administrative functions included in the project

### 📋 Menu-Driven Interface
- Simple and interactive console menu
- Easy navigation between different banking services
- Option to safely exit the application

## 🚀 How to Run

1. Open the project in VS Code or any Java IDE.
2. Compile the Java source files.
3. Run the main Java file.
4. Follow the on-screen menu to register, log in, and perform banking operations.

## 🎯 Purpose

The purpose of this project was to apply Programming Fundamentals concepts by developing a console-based banking application in Java. It was designed to simulate basic banking operations such as account registration, user authentication, deposits, withdrawals, money transfers, and loan management through an interactive menu-driven system.

This project helped strengthen my understanding of problem-solving, program design, and implementing real-world scenarios using core programming concepts.
